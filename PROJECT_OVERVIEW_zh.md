# 海上漂浮风机平台的鲁棒安全PPO控制

> **免责声明**：本项目为学习和展示目的的简化仿真复现，不声称与实际漂浮式海上风力机系统具有工程保真度。仿真使用带高斯噪声扰动的抽象质量-弹簧-阻尼模型，不可用于工程设计、安全认证或运行决策。

---

## 一、项目概览

**项目名称**（英文）：Robust and Safe PPO Control for a Simplified Floating Offshore Wind Turbine Platform

本项目构建了一个完整的深度强化学习控制管线，用于稳定受随机风浪扰动的简化漂浮式海上风机平台。项目比较了经典PD控制器与基于PPO的强化学习控制器在四种风况场景下的表现，并揭示了"免费午餐不存在"的重要科学事实。

**核心发现**：在所有分布内场景中，经典PD控制以零故障率和低7倍控制能耗的表现显著优于所有PPO变体。仅在分布外（OOD）场景中，安全过滤PPO以18%的回报优势险胜PD，但伴随着更高的故障率（40% vs 35%）和近乎恒定的安全干预。

| 场景 | 胜者 | 平均回报 | 关键发现 |
|------|------|----------|----------|
| 正常风况 | **PD** | -2.59 | PD优于所有PPO变体超过4.6倍，故障率0% |
| 强风况 | **PD** | -10.10 | PD显著领先，PPO最佳变体差3倍且故障率20-30% |
| 可变风况 | **PD** | -10.90 | PD胜出；所有PPO变体故障率约33%，回报差5倍 |
| 分布外风况 | **PPO+安全** | -30.19 | 比PD好18%，但故障率更高（40% vs 35%），样本量小（n=3种子） |

## 二、技术路线

### 2.1 问题公式化（MDP）

本项目将漂浮平台控制问题形式化为马尔可夫决策过程：

- **状态空间（4维）**：平台俯仰角 θ、俯仰角速度 θ_dot、风扰动代理、波浪扰动代理
- **动作空间（1维）**：单一连续控制力 `a ∈ [-1, 1]`，映射为 `F = a × 0.5 N·m`
- **动力学模型**：前向欧拉积分的质量-弹簧-阻尼模型：
  ```
  θ_{t+1} = θ_t + θ̇_t × dt
  θ̇_{t+1} = θ̇_t + (dt/m) × (F_wind + F_wave + F_control - cθ̇_t - kθ_t)
  ```
- **奖励函数**：`r = -1.0·θ² - 0.5·θ̇² - 0.1·F² + safety_penalty`
- **终止条件**：|θ| > 0.3 rad 时终止；最大步数 1000 步截断

### 2.2 控制器配置

| 控制器 | 描述 | 核心参数 |
|--------|------|----------|
| **NoControl** | 始终返回零动作，暴露无控动态 | 无 |
| **PD** | 经典比例-微分反馈 | Kp=5.0, Kd=2.0 |
| **PPO** | Proximal Policy Optimization | lr=3e-4, gamma=0.99, 2×64隐藏层 |
| **PPO+安全** | PPO + 一步前向预测安全过滤器 | 紧急停止阈值=0.3 rad，减速阈值=0.24 rad |
| **PPO-随机化** | PPO + 域随机化训练 | 每回合随机化物理参数 |
| **PPO-随机化+安全** | 域随机化PPO + 安全过滤器 | 组合上述两种策略 |

### 2.3 风况场景

| 场景 | 风扰动标准差 | 波扰动标准差 | 描述 |
|------|-------------|-------------|------|
| `normal_wind` | 0.3 | 0.2 | 基准运行条件 |
| `strong_wind` | 0.6 | 0.4 | 增强风浪活动 |
| `variable_wind` | 0.3 | 0.2 | 风均值从0线性递增到0.5 |
| `out_of_distribution` | 0.9 | 0.7 | 极端扰动水平，鲁棒性测试 |

## 第三轮评估：多种子扩展PPO训练（已完成）

第三轮评估解决了前两轮的两个方法论问题：（1）单种子评估无法区分真实的策略改进与随机运气；（2）5万训练步可能不足以让PPO收敛到具有竞争力的策略。

**评估协议**：3个种子（0, 1, 2），每种子50万训练步，4种风况场景，每场景20回合评估。

### 3.1 评估指标

| 指标 | 测量内容 | 好的方向 |
|------|----------|----------|
| `avg_return` | 每回合平均累积回报 | 越高（越不负面） |
| `mean_abs_theta` | 平均俯仰偏差（弧度） | 越低 |
| `failure_rate` | 触及安全限制的回合比例 | 越低（0最优） |
| `control_energy` | 动作平方和，代理执行器磨损 | 越低（视情况而定） |

### 3.2 验证结果

| 场景 | 胜者 | 证据质量 |
|------|------|----------|
| normal_wind | **PD** (-2.59) | 非常强——PD以>4.6倍优势击败所有PPO变体 |
| strong_wind | **PD** (-10.10) | 强——PD以>3倍优势领先 |
| variable_wind | **PD** (-10.90) | 非常强——PD以>5倍优势领先 |
| OOD_wind | **PPO+Safety** (-30.19) | 弱/边缘——仅优于PD 18%，故障率更高 |

### 3.3 PPO种子方差问题

种子0在所有PPO变体的可变风和OOD风场景中出现灾难性故障（100%故障率）。种子1-2有时能与PD竞争。这种双峰分布意味着单种子PPO评估结论并不可靠，揭示了RL部署的根本性风险。

### 3.4 关键发现

1. **经典PD在低维线性系统中具有解析接近最优的性质**——这并非RL的失败，而是科学上重要的结果。
2. **RL的价值在高维、非线性或多目标场景中才能充分体现**。
3. **多种子评估是RL研究的标准要求**——单种子结果可能完全误导结论。
4. **安全过滤PPO在OOD场景中显示潜力**，但边际改进需谨慎解读。

## 四、使用方法

### 安装

```bash
# 创建并激活虚拟环境
python -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 烟雾测试

```bash
# 环境烟雾测试
pytest tests/

# 或运行环境内置自测
python -m src.envs.floating_platform_env
```

### 训练

```bash
# PPO烟雾测试（1000步）
python -m src.training.train_ppo --smoke-test --scenario normal_wind

# 5万步快速实验
python -m src.training.train_ppo --timesteps 50000 --seed 42 --scenario normal_wind

# 50万步复现第三轮结果
python -m src.training.train_ppo --timesteps 500000 --seed 0 --scenario normal_wind
```

### 评估与可视化

```bash
# 评估所有控制器
python -m src.evaluation.evaluate_agents --episodes 20

# 生成结果图
python -m src.analysis.plot_results
```

## 六、证据边界

本项目灵感来源于公开研究（包括DeepSense、RL-PSF和FloatingFarmYaw），**但所有实现代码均为独立原创编写**，未从任何外部仓库复制代码。

- **DeepSense**：仅参考深度学习用于海洋/传感器数据的一般概念——所有代码原创
- **RL-PSF**：仅参考安全感知RL概念——安全过滤器独立设计
- **FloatingFarmYaw**：仅参考漂浮风电的RL概念——环境和RL公式原创

## 七、简历摘要

> 设计并评估了一个用于漂浮式海上风机平台稳定的强化学习控制管线，在3个随机种子下训练PPO至50万步。多种子分析揭示：经典PD控制在4种风况中的3种场景中以零故障率和低7倍执行器能耗优于RL——这一验证过的"没有免费午餐"发现展示了诚实的科学评估，而非强制RL优越性。实现了域随机化、安全过滤和完整的评估协议，为"RL何时是/不是控制任务的正确工具"提供了项目级证据。
